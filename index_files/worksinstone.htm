works in stone

 

This is a new entry for mid 2019:

In addition to works in print, I sometimes create things with longer projected lifespans.

All are created without the benefit of power tools.

The figures are provided here to show how these are created so if anyone is interested

in taking up this practice they can have a guide to follow.
